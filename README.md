
# nl-runtime

Write JavaScript using Japanese natural language in the browser.

## Example

```html
<script type="text/nl">
クリックしたら背景を赤にする
</script>
````

## CDN

```html
<script src="https://qianyu15.github.io/nl-runtime/main.js"></script>
```

## What is this?

A browser runtime that executes JavaScript from Japanese natural language.
