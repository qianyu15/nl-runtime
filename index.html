What is NL Runtime?

NL Runtime is an experimental browser runtime that allows developers to write simple UI interactions using Japanese natural language.

Instead of writing JavaScript event handlers directly, you can describe behavior in plain Japanese sentences.

Example

<script src="https://qianyu15.github.io/nl-runtime/main.js"></script>

<script type="text/nl">
クリックしたら背景を赤にする
</script>

<script type="text/nl">
ボタンを押したらアラートを表示する
</script>

Features

Natural Language

Write browser interactions in Japanese sentences.

Browser Only

No server and no AI model required.

Experimental

Built for learning and rapid prototyping.
